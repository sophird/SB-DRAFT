Recovery Instructions

  1. Supabase Integration: Our primary database is hosted on Supabase. Please note that automated backup triggers within this portal are currently under maintenance.
  2. Manual Export: Until the maintenance is complete, use the "Export Backup" button above to generate a manual snapshot. The file will be saved as a .json file.
  3. Secure Storage: Store the exported file in a secure location—a password-protected cloud drive or an encrypted external storage device is highly recommended.
  4. Data Restoration: To restore data after a system failure or accidental deletion, contact the lead developer or system administrator and provide the latest valid .json backup file.
  5. Advanced Recovery: For critical technical recovery, database administrators can use the Supabase CLI or PostgreSQL pg_restore tools to re-import collections from the exported data.
Recommendation: To ensure data integrity for SerbisyoBurgos, please export a manual backup at least once a week, or immediately after a large batch of resident registrations or document approvals.
